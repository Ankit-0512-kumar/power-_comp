Project: YouTube Sentiment AI Analyzer | Live Link

Developed a real-time NLP application to classify YouTube comment sentiment using a Transformer-based model (DistilBERT) and visualized insights on an interactive dashboard with Streamlit and Plotly.

Engineered a data pipeline to fetch and process hundreds of comments via the YouTube Data API, implementing pagination for scalability.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------



Interviewer: "Okay Ankit, aapne apne resume mein 'YouTube Sentiment AI Analyzer' project likha hai. Iske baare mein kuch bataiye."

Aapka Answer:

"Sir, mera project 'YouTube Sentiment AI Analyzer' ek full-stack AI web application hai. Iska main goal content creators aur brands ko ye samajhne mein help karna hai ki unki audience kisi particular video ke baare mein kya feel kar rahi hai – Positive, Negative, ya Neutral."

"Maine is project ko 3 main parts mein banaya hai:"

1. Data Collection & Processing (The Pipeline)
"Sabse pehle, data laane ke liye maine Google ki official YouTube Data API v3 ko integrate kiya hai. Ek challenge ye tha ki API ek baar mein sirf 100 comments deti hai. Is problem ko solve karne ke liye maine API Pagination ka logic likha. Mera code ek loop mein API ko baar-baar call karta hai, har baar nextPageToken ka use karke, jab tak user ki di hui limit (jaise 500 ya 1000 comments) poori na ho jaye. Isse mera application scalable ban gaya aur popular videos par bhi crash nahi hota."

2. The AI Core (Sentiment Analysis)
"Data aane ke baad, asli kaam shuru hota hai – sentiment classify karna. Iske liye maine ek simple rule-based model jaise VADER use nahi kiya. Maine Hugging Face Transformers library ka use karke ek Deep Learning model, 'DistilBERT', implement kiya."

(Yahan interviewer shayad puche: "DistilBERT hi kyun? Koi aur kyun nahi?")

Aap bologe: "Sir, YouTube comments bohot informal hote hain, usme slang, emojis aur sarcasm hota hai. DistilBERT ek Transformer model hai jo poore sentence ka context (matlab) samajhta hai, isliye wo VADER se zyada accurate hai. Maine shuru mein RoBERTa bhi try kiya tha, lekin wo thoda heavy tha free server ke liye. DistilBERT performance aur accuracy ka perfect balance tha."

3. Visualization & Deployment (The Dashboard)
"Analysis ke baad, results ko user-friendly tarike se dikhana zaroori tha. Maine Streamlit ka use karke ek interactive dashboard banaya. Isme Plotly se Pie Charts (jo sentiment ka distribution dikhate hain) aur Histograms (jo emotion ki intensity dikhate hain) add kiye."

"Finally, is poore project ko internet par live karne ke liye maine isse Streamlit Cloud par deploy kiya. Isse koi bhi mere diye hue link se is app ko use kar sakta hai. Security ke liye, maine apni API Key ko code mein daalne ki jagah Streamlit Secrets mein store kiya, jisse wo safe rahe."
